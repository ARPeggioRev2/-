# こみゃくのおと (Co-MYAKU Note)

**Version 2.0.8**

「こみゃくのおと」は、2025年日本国際博覧会（大阪・関西万博）のデザインシステム「ID」からインスパイアされた、非公式のインタラクティブ・サウンド・ファンアート（二次創作）です。

画面中央の「ID（こみゃく）」をタップすることで、自動生成されるアンビエント・ミュージック（環境音楽）と視覚的なエフェクトを楽しむことができます。

---

## ■ 主な機能

### 1. インタラクティブ・サウンド
中央のこみゃくをタップすることで演奏の開始/停止を切り替えます。

### 2. ダイナミック・ビジュアル
音楽のパート（ピアノ、ストリングス、ベース）に連動して、画面上に赤や青の「こみゃく」たちが生成されます。

### 3. 視線追従 & インタラクション
- **視線追従:** 画面の周囲をタップ・移動操作すると、中央のこみゃくがその方向を追いかけます。
- **フィードバック:** 周囲をタップした際は、波紋エフェクトと共にキラキラとした音（コンタクトサウンド）が鳴り、視覚的なフィードバックを楽しめます。

### 4. アクティビティ制御
視線の高さ（タップ位置の上下）によって、音楽の「アクティビティ（音の密度）」が僅かに変化します。

### 5. カスタマイズ機能（設定メニュー）
右上の設定アイコン（歯車に目のパーツが付いたアイコン）から以下の設定が可能です。
- **背景スタイル:** 背景を「WHITE」にするか、手持ちの写真をアップロードして背景に設定できます。
- **マスターボリューム:** 全体の音量を調整します。
- **テンポ切り替え:** 通常テンポ（120BPM相当）とハーフテンポ（60BPM相当）を選択できます。
- **パート別ミュート:** ピアノ、ストリングス、ベースの各楽器を個別にオン/オフできます。

---

## ■ 使い方

1. `index.html` をブラウザで開きます。
2. 画面中央をタップすると音楽が始まります。
3. 画面のあちこちをタップしたりなぞったりして、音と視覚の変化を楽しんでください。
4. 右上の設定アイコンをクリックすると、音量や背景の設定メニューが開きます。
