# こみゃくのおと (Co-MYAKU Note)

**Version 2.1.0**

「こみゃくのおと」は、2025年日本国際博覧会（大阪・関西万博）の公式キャラクターからインスパイアされた、非公式のインタラクティブ・サウンド・ファンアート（二次創作）です。

Web Audio APIによる自動生成アンビエント・ミュージックと、Canvas APIによる有機的なビジュアル・インタラクションを融合させた、ブラウザで体験するデジタル・トイです。

---

## ■ 主な機能

### 1. オープニング・エクスペリエンス
Ver 2.1.0より、没入感を高めるオープニング画面を実装しました。「START」ボタンを押すことでオーディオ・コンテキストが有効化され、シームレスに「こみゃくのおと」の世界が始まります。

### 2. ジェネレーティブ・オーディオ
ピアノ、ストリングス、ベースの3パートが、それぞれ異なる素数周期のアルゴリズムに従って音を奏でます。
- **視線連動:** 中央の「こみゃく」の視線の高さ（操作位置の上下）に応じて、音の密度（アクティビティ）が動的に変化します。
- **コンタクトサウンド:** 画面を操作した際、音階に沿ったフィードバック音が重なり、ユーザー自身も演奏に参加できます。

### 3. ダイナミック・ビジュアル
- **視線追従:** タッチやマウス操作に合わせて、中央の「こみゃく」が生き生きと視線を動かします。
- **パーティクル生成:** 音のトリガーに連動して、赤・青・白の「こみゃく」やバブルが画面いっぱいに広がり、音楽を視覚化します。

### 4. カスタマイズ機能（設定メニュー）
右上の設定アイコンから、体験を自分好みに調整できます。
- **背景スタイル:** 最小限の「WHITE」背景、または「PHOTO SELECT」からお気に入りの写真を背景に設定して、自分だけの展示空間を作れます。
- **マスターボリューム:** 全体の音量をスライダーで直感的に調整可能です。
- **テンポ切り替え:** 1/1 (120BPM) と 1/2 (60BPM) の切り替えにより、楽曲の表情をガラリと変えられます。
- **パート別ミュート:** 楽器ごとのオン/オフにより、ソロ演奏のような構成も楽しめます。

---

## ■ 使い方

1. ブラウザで `index.html` を開きます。
2. タイトル画面の「START」をクリックして開始します。
3. 画面をタップ、またはドラッグして、音と視覚の反応を楽しんでください。
4. 右上の設定メニューから、ボリュームや背景をカスタマイズできます。
