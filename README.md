こみゃくのおと 2.0.4
「こみゃくのおと」は、2025年日本国際博覧会（大阪・関西万博）のデザインシステム「ID」からインスパイアされた、非公式のインタラクティブ・サウンド・ファンアート（二次創作）です。
画面中央の「ID(こみゃく)」をタップすることで、自動生成されるアンビエント・ミュージック（環境音楽）と視覚的なエフェクトを楽しむことができます。
■ 主な機能
・ インタラクティブ・サウンド: タップで演奏の開始/停止を切り替えます。
・ ダイナミック・ビジュアル: 音楽のパート（ピアノ、ストリングス、ベース）に連動して、画面上に赤や青のこみゃくたちが生成されます。
・ 視線追従: マウス移動やタッチ操作に合わせてこみゃくの目が動き、視線の高さによって音楽の「アクティビティ（音の密度）」が僅かに変化します。
・ カスタマイズ機能（設定メニュー）:
　- 背景スタイル: 背景を「WHITE」にするか、手持ちの写真をアップロードして背景に設定できます。
　- マスターボリューム: 全体の音量を調整します。
　- テンポ切り替え: 通常テンポ（120BPM相当）とハーフテンポ（60BPM相当）を選択できます。
　- パート別ミュート: ピアノ、ストリングス、ベースの各楽器を個別にオン/オフできます。
■ 使い方
1.	index.html をブラウザで開きます。
2.	画面中央をタップすると音楽が始まります。
3.	右上の設定アイコン（歯車に目のパーツが付いたアイコン）をクリックすると、音量や背景の設定メニューが開きます。
